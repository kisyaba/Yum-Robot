# Yum-Robot :) Follow the below guidelines to run the project

## Installation
Follow the below to get started

```bash
git clone {repo}
```

```bash
Run npm i --save -dev @types/node
```

## Running the simulation in the terminal
 Open the repo folder in your terminal 

```bash 
cd yumRobot folder
```

```bash 
Run npm run devYum
```
